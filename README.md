# Приложение в стиле Excel

Это веб-приложение в стиле Excel, построенное с использованием современных веб-технологий. Приложение предоставляет интерфейс электронной таблицы с функциями, аналогичными традиционному программному обеспечению для работы с электронными таблицами.

## Возможности

- Секция заголовка с полем ввода и кнопками действий
- Панель инструментов для дополнительной функциональности
- Строка формул для ввода и редактирования формул
- Компонент таблицы/сетки для отображения и редактирования данных
- Адаптивный дизайн с чистым пользовательским интерфейсом

## Используемые технологии

- **Vite** - Инструменты нового поколения для быстрой разработки
- **SCSS** - Язык расширения CSS с переменными, миксинами и многим другим
- **ES Modules** - Современная система модулей JavaScript
- **normalize.css** - Библиотека нормализации CSS
- **ESLint** - Инструмент проверки кода
- **Prettier** - Инструмент форматирования кода

## Структура проекта

```
├── index.html              # Основной HTML-файл с структурой приложения
├── package.json            # Зависимости и скрипты проекта

│   ├── main.js            # Основная точка входа JavaScript
```

## Установка и разработка

1. **Предварительные требования**
   - Node.js (v14 или выше)
   - npm или yarn

2. **Установка**
   ```bash
   # Клонировать репозиторий
   git clone <repository-url>
   
   # Перейти в директорию проекта
   cd js-app
   
   # Установить зависимости
   npm install
   ```

3. **Сервер разработки**
   ```bash
   # Запустить сервер разработки
   npm run dev
   
   # Приложение будет доступно по адресу http://localhost:5173
   ```

4. **Сборка для продакшена**
   ```bash
   # Собрать проект
   npm run build
   
   # Предварительный просмотр продакшен-сборки
   npm run preview
   ```

## Архитектура SCSS

Проект использует модульную архитектуру SCSS с:

- **Переменные**: Централизованные определения переменных в `_variables.scss`
- **Миксины**: Повторно используемые шаблоны стилей в `_mixins.scss`
- **Глобальные**: Импорты переменных и миксинов в `_globals.scss`
- **Компоненты**: Отдельные файлы SCSS для каждого компонента пользовательского интерфейса
- **Индекс**: Основная точка входа, объединяющая все компоненты SCSS

## Качество кода

- ESLint с конфигурацией Airbnb-base обеспечивает последовательный стиль кода
- Prettier автоматически форматирует код в соответствии с установленными правилами
- SCSS следует модульному подходу с переменными и миксинами для удобства сопровождения

## Поддержка браузеров

Приложение использует современные функции CSS и JavaScript. Совместимо с современными браузерами, поддерживающими ES-модули и CSS Grid/Flexbox.
